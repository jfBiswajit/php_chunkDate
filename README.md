## ChunkDate

This is a helper function which helps to chunk date into days, month, years

__Example: 1__

```
Input: chunkDate('2020-01-12');
Output: 3 days ago
```

__Example: 2__

```
Input: chunkDate('2019-12-12');
Output: 1 month ago
```


__Example: 3__

```
Input: chunkDate('2016-01-12');
Output: 4 years ago
```
